# Matrix Format Converter
* Converts supported matrix formats to other formats.
* Supports Latex fraction convertion.

## Matrix Format Support:
1. Latex Matrix
```
1 & 1 & 0 & 0 & 2\\
0 & 0 & 0 & \frac{1}{4} & -3
```
2. Wolframalpha Matrix
```
{{1 , 1 , 0 , 0 , 2}, {0 , 0 , 0 , 1/4 , -3}}
```
3. IPython Array Matrix
```
[[1 , 1 , 0 , 0 , 2], [0 , 0 , 0 , 1/4 , -3]]
```
